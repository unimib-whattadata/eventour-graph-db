# Eventour Ontology

This folder contains the Eventour ontology:

- `eventour.ttl`: Turtle serialization.
- `eventour.nt`: N-Triples serialization.
- `eventour.owl`: OWL-named Turtle copy for ontology tools.

The ontology namespace is `http://eventour.unimib.it/`. Classes use UpperCamelCase; properties use lowerCamelCase; SKOS concepts use lowercase path-based identifiers.

The patched ontology contains 37 classes, 123 properties, and 593 triples.

# Portability to New Cities

Eventour is designed to be portable, but the full release is validated for Milan.

A new city profile should define:

- city identifier and label;
- base namespace;
- administrative polygons for neighborhood assignment;
- source datasets and source licenses;
- source identifier fields;
- URI templates;
- field-to-property mappings;
- controlled vocabularies;
- CRS and geometry policy;
- Wikidata city QID;
- validation gates;
- competency queries.

Porting requires manual review when source schemas, identifiers, categories, or Wikidata evidence differ from Milan.

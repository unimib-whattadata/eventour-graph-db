# Data Model

Eventour separates global ontology terms from city-scoped resources.

- Global ontology namespace: `http://eventour.unimib.it/`
- Milan namespace: `http://eventour.unimib.it/milan/`

The KG uses GeoSPARQL for features and geometries, PROV-O for provenance, DC Terms/DCAT for source metadata, SKOS for controlled vocabularies, Schema.org and LOCN for address-related descriptions, and OWL for Wikidata identity links.

Main Eventour families include administrative areas, physical assets, facilities, services, transport entities, semantic places, source records, and controlled role/category concepts.

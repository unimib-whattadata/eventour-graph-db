# Resource Availability

## Persistent Release

- DOI: `https://doi.org/10.5281/zenodo.20076141`
- GitHub release: `https://github.com/unimib-whattadata/eventour-resource/releases/tag/v1.0.0`
- SPARQL endpoint: not bundled with the repository; the release can be loaded locally using `docs/graphdb-loading.md`.
- Version: `v1.0.0`

## Resource Components

This release contains the RDF graph dump, Eventour ontology, VoID/DCAT metadata, SHACL shapes, municipal mappings, Wikidata extraction and curation artifacts, validation reports, competency queries, and application-code pointers.

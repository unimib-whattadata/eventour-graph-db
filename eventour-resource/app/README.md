# Application Code

This folder links the resource package to Eventour application code. The KG resource is maintained as the canonical citable repository; UI and GraphDB deployments may be kept as companion repositories or added as Git submodules.

- `itinerary-generation/` contains the notebooks, datasets, and integration
  artifacts for the itinerary-generation and gamified crowd-management use case.
- `graphdb/` contains GraphDB loading helpers.
- `ui/` points to the companion Eventour UI repository.

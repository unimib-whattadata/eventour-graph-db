# Data

The full Eventour Milan KG dump is distributed through Zenodo/GitHub Releases rather than stored directly in Git.

- `final/`: download instructions for the full KG.
- `sample/`: small sample data for smoke tests.
- `source-snapshots/`: source catalog and license table. Raw Comune di Milano
  source snapshots are not stored in Git; they can be downloaded from the listed
  public URLs.
- `release-manifest-v1.0.0.json`: release manifest.
- `checksums-sha256.txt`: release checksums.
